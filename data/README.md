# Data Directory

This directory contains the output files generated by the pipeline:

- PostgreSQL tables are written to the `postgres-dev` service (`marketing_db`).
- `resultados_atribuicao.parquet` — attribution results export for Power BI.
- `fato_jornadas.parquet` — journey fact export for Power BI.

> ⚠️ Parquet files are generated by the pipeline and are **not** tracked in git.
